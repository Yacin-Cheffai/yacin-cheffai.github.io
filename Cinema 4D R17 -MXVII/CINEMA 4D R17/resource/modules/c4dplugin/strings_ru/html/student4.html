Для получения кода активизации программы кликните на расположенной ниже ссылке и зарегистрируйтесь в компании MAXON.

После получения вами кода активизации по электронной почте (тема: 'Ваш CINEMA 4D код активизации для студенческой версии') кликните на кнопку 'Добавить код активизации' расположенную ниже и скопируйте / добавьте его в окно регистрации из электронной почты.

Код активизации функционирует только с тем компьютером, который вы использовали с целью регистрации / активизации программы.
